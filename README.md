# CADREB Stats Pipeline

## Purpose

Build a scalable cloud-based pipeline that collects market statistics from CADREB's 10K Research platform and stores them in a normalized database.

## Goals

- Automatically download dozens (eventually hundreds) of datasets
- Support multiple boards
- Support multiple property types
- Store all data in a normalized format
- Feed Google Sheets dashboards
- Run automatically in Google Cloud
